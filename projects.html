Drinks Movies App


This project is far from profound, but still, it is a fun app that you can use
with you friends on movie night. Using TheMovieDB and TheCocktailDB APIs, the
Drinks Movies App allows you to search any movie, and a drink will be matched,
based on the liquor type. The instructions and ingredients will be provided, so
you can concoct it yourself!


You can checkout more about this project by clicking the links below!

Live Site GitHub

EJS, PostgreSQL, Sequelize

P3


Bam


This is Bam, a utility app for martial artists. Bam includes a stopwatch,
an image board with preset demonstrations, a video board with a draw mode
that allows for video analysis, and a discussion page. With Bam, you can also
create and save training regimens to your account.
